Projetos em Ciência de Dados

Análise de perguntas sobre tecnologia no Stack Overflow!

Manipulação e visualização de dados básica sobre perguntas de tecnologia nos dados abertos do Stack Exchange Data Explorer com o objetivo de identificar quais as linguagens de programação que obtiveram um maior número de perguntas e uma maior evolução ao longo do período analisado para ter uma noção aproximada de quantas pessoas a estão usando. Ou seja, o objetivo é basicamente identificar as linguagens que tem uma comunidade muito forte, e as que estão em potencial grande de crescimento. No final da análise, eu procurei identificar linguagens de programação voltadas para análise de dados e busquei selecionar as melhores linguagens para iniciar o aprendizado em Análise de Dados.

As ferramentas utilizadas foram:

  • R e RStudio
  • ggplot2 e dplyr para visualização gráfica e manipulação de dados
